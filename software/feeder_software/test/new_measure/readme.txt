Для того чтобы начать тест, создайте папку /pi/home/test_{date}
Затем создайте папку /pi/home/test_{date}/docs

Для начала теста необходимо передать на raspberry в папку /pi/home/test_{date} следующие файлы:
1. _config.py
2. _headers.py
3. config.ini
4. test_hx_new.py
5. new_class.py

Запустить test_hx_new.py. 
Если это первый тест, рекомендую сначала произвести калибровку.
